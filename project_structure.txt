## 關鍵檔案說明

### 核心腳本
- full_batch_rename_execute.py: 使用 Qwen3-VL 30B 進行批量圖片分析
- deduplicate_and_cleanup.py: 檢測和刪除重複檔案

### 配置檔案
- environment.yml: Conda 環境定義（包含 Python 版本、LM Studio API 連接）
- config.yaml: 模型參數、API 端點、處理選項

### 文檔
- copilot-instructions.md: AI 助手的指導原則和最佳實踐
- ARCHITECTURE.md: 系統設計、資料流、技術棧
- FIRST_PRINCIPLES.md: 項目設計哲學和核心概念

### 資料流
Images → Analysis (Qwen3-VL) → JSON Results → Rename → Validation → Mapping Table
